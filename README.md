# SERVER MC

Servidor público de **Minecraft 26.1 Fabric + Mods**

---

## 📥 Paso 1: Descargar e instalar SKLauncher

1. Descarga **SKLauncher** desde su página oficial:  
   👉 https://skmedix.pl/
2. Instálalo normalmente.
3. Abre el launcher y sigue los pasos tal cual como se muestran en las imágenes de abajo.

> 🔹 **Opcional:** Puedes registrarte e iniciar sesión en SKLauncher para usar el **sistema de skins**.

## ⚙️ Paso 2: Descargar e instalar los mods automáticamente

Descarga el archivo el **script**:

👉 [Instalar-Mods.bat](https://github.com/jesusmuentes/server_mc/releases/download/v1.0/Instalar-Mods.bat)

> ℹ️ **Información:**  
El script descargará y colocará **todos los mods automáticamente** en tu carpeta `.minecraft`.  No necesitas instalar nada manualmente.

---

## 🧱 Paso 3: Seleccionar la versión y jugar

1. En **SKLauncher**, selecciona la versión que creamos arriba: **GGZone-26.1.2**
2. Haz clic en **Jugar**

---

## 🌍 Paso 4: Entrar al servidor

1. Ve a **Multijugador**
2. Haz clic en **Añadir servidor**
3. Ingresa la IP del servidor:  ⚠️ **IP TEMPORAL – NO REAL**

---

## ✅ ¡Eso es todo!

Si tienes problemas, revisa que los datos se hayan configurado **exactamente** como se muestran en las imágenes:

- **GGZone-26.1.2** escrito de forma exacta
- **Fabric** seleccionado correctamente
- **Versión 26.1.2** configurada
- **Casilla del directorio del juego** marcada  
  - El directorio debe marcarse **después** de haber colocado el nombre

Si ninguno de estos puntos es el problema y ya seguiste todos los pasos:
- Elimina la versión que creaste
- Repite el proceso desde el inicio

¡Disfruta del servidor! 🎮🚀
